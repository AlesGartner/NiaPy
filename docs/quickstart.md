# Quickstart

## Installation

We are trying very hard to provide you with the most convinient ways for you to install the NiaPy. Currently you can install it using: **pip**, **conda**, **Arch AUR package**, **Fedora package** or **Install from source**.

=== "pip"

    <div class="termy">
    ```console
    $ pip install NiaPy

    ---> 100%
    ```
    </div>

=== "conda"

    <div class="termy">
    ```console
    $ conda install -c niaorg niapy

    ---> 100%
    ```
    </div>

=== "Arch AUR package"

    <div class="termy">
    ```console
    $ yay -Syu python-niapy

    ---> 100%
    ```
    </div>

=== "Fedora package"

    <div class="termy">
    ```console
    $ yum install python3-niapy

    ---> 100%
    ```
    </div>

=== "Install from source"

    <div class="termy">
    ```console
    $ git clone https://github.com/NiaOrg/NiaPy.git

    ---> 100%

    $ cd Niapy
    $ python setup.py install

    ---> 100%
    ```
    </div>
